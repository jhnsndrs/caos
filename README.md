# CAOS Interactive Prototype

Open `index.html` in Chrome, Edge or Firefox.

This is a clickable product prototype with 14 screens, demo data, navigation, simulated forms, local browser persistence and reset functionality. It is designed as a visual and interaction specification for Base44.

## Simulated
- No real backend or authentication
- No permanent database
- No external APIs
- Simplified demo calculations

## Base44 handoff
Build this prototype as the real application while preserving the screen structure, navigation, forms, statuses, tables, warnings and KPI hierarchy. Replace local demo state with a real database and the business logic from the Excel functional master.
